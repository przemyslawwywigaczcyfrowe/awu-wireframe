# Słownik pojęć — AWU

> Wszystkie skróty i terminy używane w systemie AWU i w procesie skupu produktów używanych.

---

## Terminy procesowe

| Pojęcie | Znaczenie |
|---------|-----------|
| **AWU** | System do zarządzania skupem produktów używanych (ten, który budujemy) |
| **Ekspertyza** | Fizyczna weryfikacja stanu sprzętu przez operatora — porównanie deklaracji klienta z rzeczywistością |
| **Wycena** | Formularz złożony przez klienta (online) lub operatora (salon) z wyceną produktu/ów |
| **Negocjacja** | Proces uzgadniania nowej ceny z klientem po weryfikacji, jeśli stan produktu odbiega od deklaracji |
| **Odkup** | Proces zakupu produktu używanego od klienta |
| **Odkup pracowniczy** | Specjalny zakup od pracownika firmy — cena może być niższa niż standardowa |
| **Poranny rytuał** | Codzienna kontrola nowych umów: sprawdzenie duplikatów, brakujących danych, anulacji |
| **Refurb** | Produkt odnowiony — kupowany hurtowo od firm (np. Focus Nordica), nie od osób prywatnych |

## Dokumenty i numery

| Pojęcie | Znaczenie |
|---------|-----------|
| **Umowa kupna-sprzedaży** | Dokument podpisywany z osobą prywatną przy odkupie produktu |
| **Faktura VAT** | Dokument od firmy sprzedającej nam produkt (zamiast umowy) |
| **PCC** | Podatek od Czynności Cywilnoprawnych — naliczany od umów z osobami prywatnymi > 1000 zł. Termin: do 7. dnia następnego miesiąca |
| **CSV** | Plik z danymi ekspertyzy/umowy — generowany przez obecny system AWU i wysyłany mailem |
| **RMA** | Return Merchandise Authorization — zlecenie serwisowe w Verto |
| **Protokół pozostawienia sprzętu** | Dokument potwierdzający, jaki sprzęt klient zostawił w salonie do ekspertyzy. Klient dostaje kopię jako potwierdzenie. |
| **Protokół przyjęcia sprzętu (firma)** | Dokument dla firm — potwierdzenie przyjęcia sprzętu, na podstawie którego firma wystawia nam fakturę sprzedaży. |

## Systemy zewnętrzne

| Pojęcie | Znaczenie |
|---------|-----------|
| **Verto** | System ERP Cyfrowe.pl — **źródło prawdy** o cenach, stanach magazynowych, dokumentach zakupu |
| **Sylius** | Platforma e-commerce — sklep internetowy cyfrowe.pl |
| **Baselinker** | Narzędzie do zarządzania aukcjami na Allegro i innych marketplace'ach |
| **Cyfrolog** | System nadawania unikalnych numerów CYF i kodów EAN produktom |
| **Thulium** | Nowy system komunikacji z klientem (zastępuje OTRS). Obsługuje maile, SMS, tickety |
| **OTRS** | Stary system ticketowy do komunikacji z klientem (zastępowany przez Thulium) |

## Kanban — przygotowanie do sprzedaży

| Pojęcie | Znaczenie |
|---------|-----------|
| **Indeks** | Numer identyfikacyjny produktu w systemie Verto. Bez niego nic dalej nie ruszy — **BLOKER #1** |
| **PZ** | Przyjęcie Zewnętrzne — formalne przyjęcie towaru na stan w Verto. Bez niego produkt nie może być aktywowany na froncie — **BLOKER #2** |
| **MMK** | Przesunięcie Magazynowe — zmiana magazynu z "używany" na główny (warunek aktywacji na froncie) |
| **KGM** | Karta Gwarancyjna — indywidualny dokument gwarancji (6 miesięcy) dla każdego produktu |
| **EAN** | Kod kreskowy produktu — nadawany przez Cyfrolog |
| **Numer CYF** | Unikalny wewnętrzny numer produktu w systemie Cyfrolog |
| **Seria 03** | Numeracja dokumentów zakupu w Verto: 01 = handel, 02 = koszty operacyjne, **03 = produkty używane** |
| **Front** | Strona cyfrowe.pl — aktywacja produktu do sprzedaży |
| **Sesja** | Sesja zdjęciowa produktu — fotografowanie i profesjonalne pakowanie |
| **Karta produktu** | Listing w Sylius — opis stanu, przebieg, zdjęcia, cena |

## Stany produktu

| Pojęcie | Znaczenie |
|---------|-----------|
| **Ocena 5-10/10** | Skala stanu wizualnego i mechanicznego produktu |
| **10/10** | Stan idealny, jak nowy |
| **9/10** | Minimalne ślady użytkowania |
| **8/10** | Nieznaczne ślady użytkowania |
| **7/10** | Widoczne ślady użytkowania |
| **6/10** | Wyraźne ślady użytkowania |
| **5/10** | Bardzo mocne ślady użytkowania |
| **Rozbieżność** | Różnica między stanem zadeklarowanym przez klienta a stanem zweryfikowanym przez operatora |

## Formy płatności

| Pojęcie | Znaczenie |
|---------|-----------|
| **Przelew bankowy** | Płatność na konto klienta — standardowa forma |
| **Karta podarunkowa** | Karta do wykorzystania w sklepie Cyfrowe.pl — cena wyższa o ~5-10% |
| **Gotówka** | Płatność w salonie |
| **Przelew odwrotny** | Klient kupuje nowy sprzęt i wysyła do nas używany — rozliczenie różnicą |
| **Karta z pełną opieką** | Karta podarunkowa + pełna obsługa zamówienia nowego sprzętu |
